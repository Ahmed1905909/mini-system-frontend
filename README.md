# Mini System Frontend


## What it does?
- Login and Register
- See products
- See orders
- Dashboard with numbers

## How to start?
1. `npm install`
2. `npm run dev`

Open browser at `http://localhost:5173`.

## Important
- Backend must run on port 8000.
- If login fail, check backend.

## Code stuff
- `src/views` has pages
- `src/stores` has login logic (pinia)
- `src/components` has parts of page
